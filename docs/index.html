Welcome to the HTMLPlayground
This repo contains a variety of web pages that I’ve written, mostly in collaboration with one or more LLMs, especially Claude. There’s no rhyme or reason behind these pages — each reflects something that struck me as interesting.
How Big is a Country?
Comparison by area
Comparison by population
I was listening to a podcast and a comparison was made about the relative size of a couple of countries. I wasn’t very familiar with the countries, so I was a little surprised by the comparison. This led me to wondering about other country-country comparisons, and that started the development of this web page. Once I had it running, I thought that area is only one property to compare, so I added population. Over time I’ll probably add an economic dimension.
